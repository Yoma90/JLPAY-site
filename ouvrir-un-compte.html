Des questions ? Nous vous répondons_

Comment devenir client Push ?

Pour ouvrir un compte PUSH plusieurs possibilités s'offrent à vous:

  1. En téléchargeant l'application mobile PUSH CI disponible sur Playstore et Appstore
    • Saisir son numéro de téléphone et choisir votre code Push à 5 chiffres,
    • Joindre une photo de votre pièce d'identité en cours de validité en recto et verso
    • Renseigner vos informations personnelles
    • Valider votre inscription
    • Choisir une carte Virtuelle ou Physique pour commencer votre expérience Push
  2. En se rendant dans un point de vente PUSH muni d'une pièce d'identité en cours de validité

    3. (Se référer à la première étape)

Comment recharger une carte Push ?

Avec PUSH, vous avez la possibilité de recharger votre carte à travers 3 canaux:

  1. Depuis l'application mobile PUSH CI pour un rechargement via n'importe quel réseau mobile money.

    2. En cliquant sur le menu « Recharger une carte », sélectionnez votre opérateur puis votre numéro de téléphone mobile money suivi du montant souhaité pour le rechargement en suivant les instructions de votre opérateur mobile money.

  2. Depuis le *755#, valable quelque soit votre opérateur téléphonique, vous choisissez l'option 1- Rechargement carte Push, sélectionnez votre opérateur puis vottre numéro de téléphone mobile money suivi du montant souhaité pour le rechargement en suivant les instructions de votre opérateur mobile money.
  3. Dans tous nos points de vente Push muni de votre téléphone.
Comment retirer de l’argent avec une carte Push ?

Pour les retraits, plusieurs choix s'offrent à vous:

  1. En vous rendant dans l'un de nos points de vente Push muni de votre téléphone quelque soit la carte Push que vous possédez
  2. Dans tous les guichets automatiques (uniquement pour les détenteurs de la Carte Push)
